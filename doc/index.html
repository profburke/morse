Module morse

morse, a Lua module for translating text to Morse code.

This module provides a text to Morse code function. The one interesting bit is that the implemetation is designed to allow you to easily override the way the symbols are generated so that you could, for example, have it beep, rather than print dots and dashes.

Usage:

     morse = require 'morse'
 morse.send 'a patient waiter is no loser'

send (msg)
Translates message to Morse code.

This message is called for its side effects which vary depending on the implemenation of the signal handlers. The default implementation prints dots and dashes to standard out.

Parameters:

  • msg string the text to translate
